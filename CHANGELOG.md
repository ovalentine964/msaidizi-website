## [0.1.0] - 2026-07-02

### Added
- **15 intelligence products** documented across 4 buyer segments
- **FMCG Intelligence** product page (Pwani Oil pilot)
- **Supply Chain** and **Research Data** product listings
- **Msaidizi feature updates** — gamification, wealth mindset, tithe & giving, goals & loans
- **14 dialect support** documentation
- **Bilingual content** — 111 English/Swahili translation pairs
- **PWA support** — manifest.json, service worker, offline capability
- **SEO optimization** — Schema.org structured data, Open Graph, Twitter Cards
- **Accessibility** — Skip links, focus-visible, reduced motion support
- **Mobile-first responsive design** with sticky download bar
- **One-click APK download** from GitHub Releases
- **Floating WhatsApp contact button**
- **Newsletter subscription form**
- **Community section** — WhatsApp group, Migori pilot program
- **Install guide** — 3-step simplified instructions for non-technical users
- **Trust signals** — Founder credentials, privacy badges, Kenya-built

### Fixed
- **Swahili translations** — corrected 4 translation issues
- **Direct APK download link** — points to GitHub release asset (not releases page)
- **CSS/HTML class name mismatch** — complete BEM convention alignment
- **CSS rewrite** — all classes now match HTML structure
- **App size disclosure** — honest 15MB app + 80MB voice models
- **Security** — CSP headers added, console.log of email removed

### Changed
- **CSS minified** — 30KB → 22KB for faster load
- **JS minified** — 7KB → 4KB for faster load
- **Corporate redesign** — AfricaNenda-inspired professional layout
- **Worker-focused restructure** — removed premature B2B content
- **Africa branding** — continent silhouette in hero banner
- **Logo redesign** — African identity elements
